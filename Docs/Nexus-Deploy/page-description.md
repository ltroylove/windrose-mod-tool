# BlackFlag Mod Manager

---

## Description

BlackFlag Mod Manager is a free, open-source desktop tool for managing and creating mods for Windrose on Windows. It gives you a simple interface to install, enable, and disable .pak mods for your game client and dedicated server, and lets you generate your own custom Game Tuning mods without touching a single file.

This is our first modding project — not everything may work perfectly yet and we are actively improving it. If something breaks, please leave a bug report and we will get on it.

---

## Installation Instructions

1. Download `BlackFlagModManager.exe` from the Files tab
2. Place it anywhere on your PC and run it — no installer needed
3. The tool will auto-detect your Windrose installation via Steam
4. If it does not find it automatically, open the Settings tab and set your game path manually

---

## Main Features

**Mod Manager**
Install, enable, and disable .pak mods for both the Windrose game client and your dedicated server from a single interface.

**Game Tuning Creator**
Generate your own custom .pak mods using simple sliders — no files to edit, no modding knowledge needed. Adjust:
- Stack sizes — how many items fit in a single inventory slot
- Loot drop rates — fiber, stone, sulfur, clay, salt, herbs, crops, fishing, scrap piles, animal drops, and more
- Backpack slot count
- Fast-travel bell limit
- Lantern burn duration

(For features that need binary asset edits — base wood-per-chop, base ore-per-dig, ancient-debris spawner timing — install the dedicated community mods on Nexus.)

**Mod Library**
A unified library of all your downloaded and generated mods. Install any of them to the client or server with one click.

**Backup System**
Automatically backs up your ~mods/ folder before any install or removal so you can always roll back.

**Activity Log**
A persistent log of every install, remove, enable, disable, and generation — so you always know what changed and when.

**Dashboard**
Quick-access buttons for common actions, recent activity at a glance, and a one-click game launch button.
