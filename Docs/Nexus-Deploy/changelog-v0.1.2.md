# v0.1.2

- **Generated Game Tuning mods no longer crash the game.**
- **New Manage Remote dialog** — list and bulk-delete mods on your dedicated server via FTP.
- Various Library tab fixes (deployed badges on disabled paks, accurate mod names).

**Scope note:** Game Tuning covers stack sizes, loot drops, backpack slots, fast-travel bells, and lantern duration. For wood-per-chop, ore-per-dig, or resource respawn timing, install the dedicated community mods (MoreTreeResources, MoreMineralResources, etc.) — this manager will deploy them to client and server for you.
